# 开放docker的rest
```
vim /usr/lib/systemd/system/docker.service
systemctl daemon-reload
systemctl restart docker
ss -ntulp|grep 2375
```

# proto
protoc --go_out=./ --go-grpc_out=./  hello.proto

source /etc/environment
软链到/usr/bin

规定owner和default为第一个和第二个角色，不可更改
